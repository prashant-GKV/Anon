## ✨ Features

- **Fully Responsive Design** — looks great on desktop, tablet, and mobile devices
- **Sticky Navigation Bar** — with category dropdown menus for easy browsing
- **Mobile Slide-in Menu** — smooth hamburger menu with accordion category lists
- **Newsletter Modal** — popup subscription form to capture user emails
- **Notification Toast** — real-time style purchase notifications
- **Hero Banner Slider** — eye-catching promotional banners
- **Product Showcase** — new arrivals, trending items, top-rated & deal-of-the-day sections
- **Deal Countdown** — special offer section with discount highlights
- **Product Categories** — electronics, men's & women's fashion, jewellery, cosmetics & more
- **Testimonials & Blog Section** — build trust and engagement with customers
- **Modern UI/UX** — clean layout, smooth hover effects, and Ionicons integration

## 🚀 Getting Started

No build tools or dependencies required — just clone and open!

### Prerequisites

Any modern web browser (Chrome, Firefox, Edge, Safari) and [Git](https://git-scm.com/downloads) installed.

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/prashant-GKV/Anon.git

# 2. Navigate to the project folder
cd Anon/Anon-eCommerce-website

# 3. Open in your browser
# Simply double-click index.html, or run:
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

> 💡 **Tip:** For the best experience, use a local dev server like the **Live Server** extension in VS Code.
